INSTALLATION ON WINDOW

1. Go to this website: https://git-scm.com/downloads
2. Click on the rounded button
3. Open the .exe file
4. Click Next
5. Choose the folder for installation
6. Click Next
7. Choose Use the OpenSSL library and Next
8. Choose Check-out window style and Next
9. Choose Use Min TTY and Next
10. Choose Default and Next
11. Choose Git Credential Manager Core and Next
12. Check enable Style System Caching and Next 
13. Click Install and wait for installation
14. Click Finish

INSTALLATION 
ON MACOS

1. Go to this website: https://git-scm.com/downloads
2. Click on download for Mac
3. Click on install homebrew
4. Copy the command to the terminal
5. Enter the computer password
6. Press Enter 
7. Try some Git commands

COMMANDS

git config -l

This command is to list everything about git on your device

COMMANDS

git config --global user.name <username>

git config --global user.email <useremail>

This command is to set up user name and user email, which are used for GitHub registeration.

COMMANDS

git clone <URL>

This command is to clone into the repositories in GitHub for modifying and working. The URL can be coppied from the GitHub repositories.

COMMANDS

git add .
(git add <filename>)
git commit -m"<message>"
git push

Those commands are to add the file into the repositories with the comment of the editor. The git push command is to push the added things and commit in to GitHub for further activities.

COMMANDS

git pull

This command is to avoid overwriting as while working on the project, there sometimes occur this situation.

COMMANDS

git branch <branch name>
git checkout <branch name>

The first command is to create new branch with the name is after the branch

The second command is to move to another branch
